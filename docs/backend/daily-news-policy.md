# MaeMoJi 일일 뉴스 노출 정책

## 날짜 기준

- 모든 뉴스 날짜는 서버 위치와 무관하게 `Asia/Seoul` 기준으로 계산한다.
- 월요일부터 금요일까지는 당일 발행된 관련 뉴스만 최대 3개 노출한다.
- 평일에 당일 관련 뉴스가 없으면 오래된 뉴스를 억지로 채우지 않는다.

## 주말 처리

- 토요일과 일요일에 당일 뉴스가 있으면 해당 날짜의 뉴스를 사용한다.
- 당일 뉴스가 없으면 가장 최근 날짜의 관련 뉴스를 최대 3개 노출한다.
- 따라서 주말에는 금요일 뉴스가 유지될 수 있지만, 다음 평일에는 다시 당일 뉴스만 표시된다.

## 분석 캐시

- Gemini 분석 캐시 해시에 실제 뉴스 표시 날짜를 포함한다.
- 같은 기사는 같은 날짜 안에서 재분석하지 않되, 날짜가 바뀌면 이전 날짜 분석이 섞이지 않는다.
